# Calculadora_distribuida

OPERAÇÕES 

operações basicas:

soma
subtracao
divisao
multiplicação

operações especiais:

porecentagem
exponenciacao
radiciação

SINTAXE

funcionamento das operações :

+ soma n1 com n2
- subtrai n2 de n1
* multiplaca n1 e n2
/ divide n1 por n2
% n1 por cento de n2
^ n1 elevado por n2 
√ n1 indice da raiz de n2
